# CourtFlow Static HTML Prototype

This folder contains a static design prototype for the pickleball booking and liveboard system.

## Open this first

```txt
index.html
```

## Prototype Pages

- `index.html` — landing / product overview
- `player-booking.html` — player booking and checkout concept
- `admin-dashboard.html` — admin dashboard
- `admin-liveboard.html` — admin live court control
- `tv-liveboard.html` — large TV liveboard screen

## Notes

- This is static HTML/CSS/JS.
- No backend is connected.
- Timers are simulated with simple JavaScript.
- Player avatars are placeholders.
- The real app should later be built with Next.js + NestJS + PostgreSQL + Socket.IO.
