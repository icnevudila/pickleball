# Visual Language

## Visual Direction

The interface should look like a mix of:

```txt
Sports broadcast scoreboard
+
Premium SaaS dashboard
+
Live operations control panel
```

## Theme

### Primary Theme

Dark sports theme.

Use for:

- TV liveboard
- Admin live operation
- Design system
- Live status components

### Secondary Theme

Clean card-based dashboard.

Use for:

- Player booking
- Checkout
- Profile
- Admin tables

## Color Palette

### Background

```txt
Deep Navy: #071014
Night Blue: #091A21
Graphite: #111820
```

### Primary Accent

```txt
Pickle Green: #B7F34B
```

Used for:

- Available status
- Primary buttons
- Live indicator
- Successful states
- Next-up highlight

### Secondary Accent

```txt
Electric Cyan: #25D0FF
```

Used for:

- Realtime states
- Playing status
- Links
- Progress
- Active selection

### Warning

```txt
Timer Yellow: #FFCF5A
```

Used for:

- Ending soon
- Few spots left
- Payment pending

### Danger

```txt
Alert Pink/Red: #FF5C7A
```

Used for:

- Time up
- Cancelled
- No-show
- Court closed
- Destructive actions

### Success

```txt
Court Green: #43F29A
```

Used for:

- Paid
- Checked in
- Confirmed
- Available court

## Typography

Recommended stack:

```txt
Inter, system-ui, sans-serif
```

### TV Typography

TV typography must be big.

Recommended sizes:

```txt
TV title: 56-80px
Court name: 34-48px
Timer: 72-110px
Player name: 22-34px
Queue item: 18-28px
```

### Admin Typography

Recommended sizes:

```txt
Page title: 36-52px
Section title: 24-34px
Table text: 14-16px
Button text: 14-16px
Status badge: 11-13px
```

### Player Typography

Recommended sizes:

```txt
Mobile page title: 32-42px
Card title: 22-30px
Body: 15-17px
Button: 15-16px
```

## Shape Language

Use rounded cards.

Recommended radius:

```txt
Small badge: 999px
Input: 14-18px
Card: 22-32px
TV court card: 30-40px
```

## Shadow and Depth

Use subtle shadows for dashboard.

Use glow effects carefully on TV screen.

Allowed:

```txt
Soft neon glow around live/active areas
```

Avoid:

```txt
Heavy gamer neon everywhere
```

## Icons

Recommended icon style:

- Line icons
- Rounded corners
- Simple sport-related icons
- No overly detailed icons

## Avatar Style

Player avatars are core to the product.

Avatar rules:

- Circular or rounded-square
- Clear fallback initials
- Consistent size per screen
- Display name beside avatar
- Use team color/status border when playing

## Status Badge Style

Badges should be short and obvious.

Examples:

```txt
LIVE
PLAYING
AVAILABLE
NEXT UP
WAITING
PAID
UNPAID
TIME UP
```

## Animation Style

Animations should communicate state, not decorate randomly.

Good animations:

- Live pulse
- Timer warning pulse
- Queue item slides into next-up
- Court card changes status
- Reconnecting indicator

Bad animations:

- Constant bouncing
- Overuse of confetti
- Long transitions
- Distracting background motion

## Visual Rule

TV design can be dramatic.

Admin design must be practical.

Player booking must be frictionless.
