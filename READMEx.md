# Pickleball UI/UX Design Pack

This ZIP contains separate UI/UX Markdown documentation and a static HTML design prototype.

## Folders

```txt

wireframes/

```

## Notes

The prototype is static. It does not include backend, real payment, database, or websocket.
It is intended for visual direction and UI/UX handoff.
