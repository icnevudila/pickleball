# PadelOS Product Package

A complete product planning and screen mockup package for a padel club operating system.

## What is inside

- `/docs` — product strategy, UX flows, backend plan, database model, API outline, roadmap, acceptance criteria and copy deck.
- `/design` — design tokens and component guidance.
- `/screens/web` — desktop/web mock screens for every core page.
- `/screens/mobile` — mobile mock screens for every core page.
- `/screens/index.html` — screen directory to open all mockups quickly.

## Product thesis

PadelOS is not just a reservation tool. It is the live operating system for modern padel clubs: bookings, live courts, queues, open matches, payments and player community in one place.

## Current visual direction

Coral + cream theme with a professional sports-SaaS component system:

- Warm cream surfaces for premium club feeling
- Coral as primary brand/action color
- Amber as energy accent
- Green for active/healthy states
- Graphite text for trust and clarity

## Recommended implementation stack

The technical plan assumes a Next.js web app, Postgres/Supabase for the core data layer and realtime court updates, and Stripe for payment flows. This is a starting point, not a hard dependency.
